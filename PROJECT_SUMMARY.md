## 🚀 Quick Start (5 Steps)

### Step 1: Push to GitHub
```bash
cd readgenius-project
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/readgenius.git
git push -u origin main
```

### Step 2: Deploy Website (Choose ONE)
- **Vercel** (easiest): vercel.com → import repo → deploy
- **GitHub Pages**: Settings → Pages → enable
- **Netlify**: netlify.com → import repo → deploy

### Step 3: Get Free Domain
- **Freenom**: freenom.com → register .tk domain (FREE)
- **Or**: Namecheap (~$0.88/year first year)

### Step 4: Deploy Backend
```bash
gcloud run deploy readgenius-api \
  --source . \
  --platform managed \
  --region us-central1 \
  --allow-unauthenticated \
  --set-env-vars GEMINI_API_KEY=your-key
```

### Step 5: Apply to Google Cloud
- Go to: https://cloud.google.com/startup/apply
- Fill out form with your info
- Submit!
