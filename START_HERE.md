### Step 3: Deploy Website (10 minutes)
Choose ONE:
- **Vercel** (easiest): https://vercel.com → import repo → deploy
- **GitHub Pages**: Settings → Pages → enable
- **Netlify**: https://netlify.com → import repo → deploy

### Step 4: Get Free Domain (5 minutes)
- **Freenom**: https://www.freenom.com → register .tk domain (FREE)
- **Or**: Namecheap (~$0.88/year)

### Step 5: Deploy Backend (10 minutes)
```bash
gcloud run deploy readgenius-api \
  --source . \
  --platform managed \
  --region us-central1 \
  --allow-unauthenticated \
  --set-env-vars GEMINI_API_KEY=your-key
```

### Step 6: Prepare Application (1-2 hours)
Follow: `APPLICATION_CHECKLIST.md`

### Step 7: Submit to Google Cloud
Go to: https://cloud.google.com/startup/apply
